# [리눅스] Bash cat 사용법

## 개요
`cat` 명령어는 "concatenate"의 줄임말로, 파일의 내용을 출력하거나 여러 파일을 결합하는 데 사용됩니다. 주로 텍스트 파일의 내용을 화면에 표시하거나, 파일을 다른 파일로 복사하는 데 유용합니다. `cat`은 간단하지만 강력한 도구로, 파일을 신속하게 확인하고 조작할 수 있게 해줍니다.

## 사용법
기본적인 `cat` 명령어의 구문은 다음과 같습니다:

```bash
cat [옵션] [파일...]
```

### 일반적인 옵션
- `-n`: 출력할 때 각 행에 번호를 매깁니다.
- `-b`: 빈 행을 제외하고 각 행에 번호를 매깁니다.
- `-E`: 각 행의 끝에 `$` 기호를 추가하여 표시합니다.
- `-s`: 연속된 빈 행을 하나의 빈 행으로 압축합니다.

## 예제
1. **파일 내용 출력하기**  
   특정 파일의 내용을 화면에 출력하려면 다음과 같이 입력합니다:

   ```bash
   cat example.txt
   ```

2. **여러 파일 결합하기**  
   두 개 이상의 파일을 결합하여 출력하려면 다음과 같이 입력합니다:

   ```bash
   cat file1.txt file2.txt > combined.txt
   ```

   위 명령어는 `file1.txt`와 `file2.txt`의 내용을 결합하여 `combined.txt`라는 새로운 파일로 저장합니다.

## 팁
- `cat` 명령어는 파일의 내용을 확인할 때 유용하지만, 큰 파일을 출력할 경우 화면이 넘칠 수 있습니다. 이럴 때는 `less`나 `more`와 같은 페이지네이션 도구와 함께 사용하는 것이 좋습니다. 예를 들어:

  ```bash
  cat largefile.txt | less
  ```

- `cat`을 사용하여 파일을 결합할 때, 출력 파일이 이미 존재하는 경우 주의해야 합니다. `>` 연산자는 기존 파일을 덮어쓰므로, 데이터 손실을 방지하려면 `>>` 연산자를 사용하여 내용을 추가하는 것이 좋습니다.

이와 같이 `cat` 명령어는 파일을 다루는 데 있어 매우 유용한 도구입니다. 다양한 옵션을 활용하여 필요에 맞게 사용해 보세요.
