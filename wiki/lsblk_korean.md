# [리눅스] Bash lsblk 사용법

## 개요
`lsblk` 명령어는 리눅스 시스템에서 블록 장치의 정보를 나열하는 데 사용됩니다. 이 명령어는 시스템에 연결된 모든 블록 장치(예: 하드 드라이브, SSD, USB 드라이브 등)의 구조와 상태를 시각적으로 보여줍니다. `lsblk`는 장치의 이름, 크기, 유형, 마운트 포인트 등의 정보를 제공하여 시스템 관리자가 장치의 상태를 쉽게 파악할 수 있도록 돕습니다.

## 사용법
기본 구문은 다음과 같습니다:

```bash
lsblk [옵션]
```

### 일반적인 옵션
- `-a`, `--all`: 모든 장치를 나열합니다. 마운트되지 않은 장치도 포함됩니다.
- `-f`, `--fs`: 파일 시스템 정보를 포함하여 장치 정보를 표시합니다.
- `-o`, `--output`: 출력할 열을 사용자 지정합니다. 예를 들어, `lsblk -o NAME,SIZE,TYPE`와 같이 사용할 수 있습니다.
- `-p`, `--paths`: 장치의 전체 경로를 표시합니다.
- `-n`, `--noheadings`: 헤더 없이 출력합니다.

## 예제
1. 기본 사용법:
   ```bash
   lsblk
   ```
   이 명령어는 시스템에 연결된 블록 장치의 목록을 출력합니다.

2. 파일 시스템 정보 포함:
   ```bash
   lsblk -f
   ```
   이 명령어는 각 블록 장치에 대한 파일 시스템 정보를 포함하여 출력합니다.

## 팁
- `lsblk` 명령어는 `grep`과 함께 사용하여 특정 장치나 유형을 필터링할 수 있습니다. 예를 들어, 특정 이름을 가진 장치만 보려면 다음과 같이 사용할 수 있습니다:
  ```bash
  lsblk | grep sda
  ```
- `lsblk`의 출력 형식을 사용자 정의하려면 `-o` 옵션을 활용하여 필요한 정보만 선택적으로 표시할 수 있습니다. 이를 통해 출력 결과를 보다 간결하게 만들 수 있습니다.
- `lsblk` 명령어는 시스템의 블록 장치 상태를 빠르게 확인할 수 있는 유용한 도구이므로, 정기적으로 시스템 상태를 점검하는 데 활용하는 것이 좋습니다.
