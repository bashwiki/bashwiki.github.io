# [리눅스] Bash cp 사용법

## Обзор
Команда `cp` в Bash используется для копирования файлов и каталогов. Основная цель этой команды — создать один или несколько новых файлов или каталогов, которые являются точными копиями существующих. Это полезно для резервного копирования данных, перемещения файлов между директориями и создания дубликатов.

## Использование
Основной синтаксис команды `cp` выглядит следующим образом:

```bash
cp [опции] источник назначение
```

### Общие опции:
- `-r` или `--recursive`: Копирует каталоги рекурсивно, включая все их содержимое.
- `-i` или `--interactive`: Запрашивает подтверждение перед перезаписью существующих файлов.
- `-u` или `--update`: Копирует файлы только в том случае, если источник новее, чем назначение, или если назначение отсутствует.
- `-v` или `--verbose`: Выводит информацию о процессе копирования, показывая, какие файлы копируются.

## Примеры
### Пример 1: Копирование файла
Чтобы скопировать файл `file1.txt` в файл `file2.txt`, используйте следующую команду:

```bash
cp file1.txt file2.txt
```

### Пример 2: Рекурсивное копирование каталога
Чтобы скопировать каталог `my_directory` в новый каталог `my_directory_backup`, выполните команду:

```bash
cp -r my_directory my_directory_backup
```

## Советы
- Используйте опцию `-i`, чтобы избежать случайной перезаписи файлов. Это особенно полезно, если вы работаете с важными данными.
- При копировании больших каталогов с множеством файлов, опция `-v` поможет вам отслеживать процесс и убедиться, что все файлы были успешно скопированы.
- Если вам нужно копировать только измененные файлы, рассмотрите возможность использования опции `-u`, чтобы избежать ненужного копирования.
