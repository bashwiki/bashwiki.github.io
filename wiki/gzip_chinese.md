# [리눅스] Bash gzip 사용법

## 概述
`gzip` 是一个用于文件压缩的命令行工具，主要用于减少文件的大小，以便节省存储空间或加快文件传输速度。它使用 DEFLATE 算法进行压缩，通常用于压缩文本文件，如日志文件和源代码文件。压缩后的文件扩展名通常为 `.gz`。

## 用法
基本语法如下：
```
gzip [选项] [文件...]
```

常用选项包括：
- `-d` 或 `--decompress`：解压缩文件。
- `-k` 或 `--keep`：在压缩或解压缩时保留原始文件。
- `-v` 或 `--verbose`：显示详细的压缩过程信息。
- `-r` 或 `--recursive`：递归地压缩目录中的所有文件。

## 示例
1. 压缩文件：
   ```bash
   gzip example.txt
   ```
   这条命令会将 `example.txt` 压缩为 `example.txt.gz`，并删除原始文件。

2. 解压缩文件：
   ```bash
   gzip -d example.txt.gz
   ```
   这条命令会将 `example.txt.gz` 解压缩回 `example.txt`。

## 提示
- 使用 `-k` 选项可以在压缩文件时保留原始文件，便于在需要时进行比较。
- 在处理大量文件时，可以结合使用 `find` 命令与 `gzip`，例如：
  ```bash
  find . -name "*.log" -exec gzip {} \;
  ```
  这条命令会压缩当前目录及其子目录下所有的 `.log` 文件。
- 定期检查压缩文件的完整性，使用 `gunzip -t` 命令可以测试压缩文件的有效性。
